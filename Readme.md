
# Chatbot

A LLM based pyschological generative AI chatbot.


## Tech Stack

**Language:** Python

**Libraries:** Gradio, OpenAI





## Deployment

To deploy this project run

```bash
  gradio deploy
```
## Run
To run this program type in shell or terminal
``` bash
  python app.py
```
## API
Set openai api key

```bash
  openai.api_key = "API"

```
